### Hommachi.Stats について
大阪市の本町周辺にて，1ヶ月に1回くらいのペースで統計分析を学ぶグループです。  
StanとRを使ったベイズ統計分析をみんなで学んでいく予定です。  
よりR自体に興味がある方はKobe.Rにもぜひご参加を。  

### 対象者になる人・ならない人

* 社会人大学院に行ってる，行った，興味がある人。  
* こういうことをいっしょに話題にできる人が周りにおらず，さみしい思いをしている人。
